Detailed Description

LED indicator control and visual feedback system.

This module manages LED indicators for system status visualization. Provides different LED patterns for various operating modes including standby, active (charging), complete, and error states. Features configurable blink rates, sequential LED animations, and synchronized timing control for consistent visual feedback.
