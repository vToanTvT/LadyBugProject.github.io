Detailed Description

Public macro definitions.

Type definitions used in Power Control module.

Power control instance structure with derating information.

This structure maintains all state for power derating calculation including:

  • Current power output and derated limit
  • Derating coefficients from temperature and voltage
  • Measurement buffers for filtering (5 samples each)
  • Statistical values (average, median)
  • Update timing control
  • Hysteresis zone tracking for stable mode switching
