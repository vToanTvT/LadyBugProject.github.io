Detailed Description

Single derating point with hysteresis thresholds.

Defines a derating zone with:

  • Upper threshold (for upward crossing detection)
  • Lower threshold (for downward crossing detection)
  • Derating coefficient applied when in this zone Hysteresis prevents oscillation between adjacent zones.
