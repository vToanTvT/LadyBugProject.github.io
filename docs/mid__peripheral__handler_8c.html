Detailed Description

Implementation file for Peripheral Middleware module.

This file implements board peripheral control functions. Features include:

  • Relay control for LLC and PFC stages
  • PFC voltage compensation signal control
  • Intelligent fan speed control with multi-factor decision:
    • Temperature-based control with hysteresis (40-65Â°C)
    • AC voltage-based control with hysteresis (120-140V)
    • Combined logic for optimal cooling
  • Periodic update mechanism (10s interval)
  • Q15 format threshold conversion for efficient comparison
