Detailed Description

Control algorithms including PI and Multi-Resonant controllers.

This module provides middleware-level control algorithms for feedback control systems. It implements PI (Proportional-Integral) controllers and Multi-Resonant controllers using fixed-point arithmetic. The module supports anti-windup mechanisms, configurable limits, and multiple resonant frequency components for advanced harmonic compensation.

void controller_anti_windup_ ( S_PI_TypeDef * pController)

Apply anti-windup to PI controller.

Implements anti-windup by limiting output within configured bounds and back-calculating integral term to prevent accumulation during saturation. Prevents integral windup phenomenon.

Parameters
[in,out]pControllerPointer to S_PI_TypeDef structure.
Return values
None
Note
Call after controller_process_pi_() to limit output and adjust integral term. Essential for stable operation when output saturates.

void controller_process_pi_ ( S_PI_TypeDef * pController,
FunctionalState Status )

Process PI controller computation.

Executes PI control algorithm: computes error, proportional term, integral term, and updates output. Should be called periodically at the control sampling rate. When disabled, resets all terms to zero.

Parameters
[in,out]pControllerPointer to S_PI_TypeDef structure.
[in]StatusENABLE to run controller, DISABLE to reset.
Return values
None
Note
Call periodically to update control output. Use feedback_val field to provide measurement before calling. Anti-windup should be applied after this function using controller_anti_windup_().
