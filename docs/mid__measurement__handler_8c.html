Detailed Description

Implementation file for Measurement Middleware module.

This file implements ADC measurement processing and filtering. Features include:

  • Raw ADC value acquisition and storage
  • Dual conversion paths: Q15 fixed-point and direct float
  • Moving average filtering for noise reduction
  • Configurable low-pass filtering (multiple cutoff frequencies)
  • Periodic averaging update control (20ms default)
  • Multi-channel measurement support (6 channels):
    • Thermal sensors (PFC, LLC) - float calculation
    • Battery current - Q15 calculation
    • Battery voltage - Q15 calculation
    • DC bus voltage - Q15 calculation
    • AC input voltage - float calculation
  • Calibration support with gain and offset parameters
  • Circular buffer management for moving averages

Initialize measurement structure.

Configures all parameters for a measurement channel:

  • Zeros raw and calculated values
  • Sets calibration offset (Q15)
  • Sets scale factor for Q15 conversion (Q31)
  • Configures LPF coefficient
  • Sets software high limit for normalization
  • Sets float gain and offset for direct calculation
  • Selects calculation method (DIRECT or Q15)
