Detailed Description

LLC resonant converter control and regulation.

This module implements control algorithms for LLC resonant converter including voltage/current regulation, soft-start, and hybrid control modes (PFM/PWM). Features PI and Multi-Resonant controllers with configurable parameters, adaptive mode switching, power derate support, and comprehensive parameter definitions in Q15 fixed-point format.

LLC module function declarations.

  • Voltage PI controller with KP_VOL_Q15, KI_VOL_Q15, reference and limits
  • Current PI controller with KP_CUR_Q15, KI_CUR_Q15, reference and limits
  • Duty PI controller with DUT_KP_CUR_Q15, DUT_KI_CUR_Q15, reference and limits
  • Two resonant elements at 120Hz and 100Hz with respective gains and bandwidths
  • Multi-resonant controller with 2 resonant elements
  • Initial mode to PWM

◆ llc_regulator_current_loop_()

void llc_regulator_current_loop_ ( void )

Execute current regulation loop (PFM mode).

Reads battery current, runs PI controller, applies anti-windup, converts output to frequency (0-800kHz), sets fixed deadtime (180ns). Control flow:

  1. Read g_bat_current_inst.value_q15 as feedback
  2. Execute PI controller (ENABLE)
  3. Apply anti-windup to prevent integral saturation
  4. Convert PI output (Q15) to frequency using FREQ_MAX scaling
  5. Set LLC_END_DEADTIME (180ns) Used for current-controlled operation with frequency modulation.
Parameters
None
Return values
None
Note

void llc_regulator_duty_loop ( void )

Execute duty cycle regulation loop (PWM mode).

Reads battery current, runs PI controller with duty-specific parameters, converts output to duty cycle (0-100%). Control flow:

  1. Read g_bat_current_inst.value_q15 as feedback

void llc_soft_start_ ( void )

Execute LLC soft start sequence.

Implements two-phase soft start: Phase 1: Ramps deadtime down to target (180ns) at SOFT_START_RATE_DEA. Decrements by SOFT_START_DEADTIME_STEP every ~150 cycles (~0.1s). Phase 2: Ramps frequency down to target (170kHz) at SOFT_START_RATE_FRE. Decrements by SOFT_START_FREQ_STEP every ~250 cycles (~0.1s). Ensures smooth converter startup without inrush current. Enables PWM output during soft start.
