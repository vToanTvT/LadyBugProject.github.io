Detailed Description

Implementation file for LLC Middleware module.

This file implements control algorithms for LLC resonant converter. Features include:

  • Multiple regulation modes: PFM (Pulse Frequency Modulation) and PWM (Pulse Width Modulation)
  • Multiple control loops: current loop, voltage loop, duty loop, and cascaded voltage-duty loop
  • Hybrid control with automatic mode switching based on saturation
  • PI and Multi-Resonant controllers for improved harmonic rejection
  • Soft-start sequence with gradual deadtime and frequency ramping
  • Power derate integration for thermal management
  • Adaptive current threshold based on voltage level
  • Fixed-point Q15 arithmetic for efficient embedded execution
