Detailed Description

Global definitions, timing configurations, and ramp utilities.

This module provides system-wide definitions including sampling time configuration, timer handle references, and utility functions for ramping operations on fixed-point values. Used across multiple middleware modules for consistent timing and signal processing.

Ramp down a value towards target.

Gradually decreases a UQ31 fixed-point value by a fixed step size until it reaches the target value. Used for smooth transitions in control parameters to avoid abrupt changes.

Parameters
[in]targetTarget value to reach (UQ31 format).
[in]stepDecrement step size per call (UQ31 format).
[in,out]instPointer to current value being ramped (UQ31 format).
Return values
None
Note
Value is clamped at target when reached. Call repeatedly at fixed rate for consistent ramping behavior.

uq31_t start,
uq31_t target,
uq31_t rate_x_dt )

Ramp up with rate control.

Performs rate-controlled ramp-up from start to target value using specified rate. On first call, initializes to start value. On subsequent calls, increments by rate*dt until target is reached. Uses static initialization flag for first-call detection.

Parameters
[in,out]instPointer to current value being ramped (UQ15 format).
[in]startInitial starting value (UQ31 format).
[in]targetFinal target value to reach (UQ31 format).
[in]rate_x_dtIncrement per call (rate * sampling time, UQ31).
Return values
None
Note
Uses static variable for initialization tracking. Not reentrant. Only ramps upward (monotonic increase) towards target.
