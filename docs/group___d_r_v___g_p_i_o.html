Detailed Description

GPIO driver for digital pin manipulation and control.

This file implements the initialization and main process logic of the GPIO Driver module. It provides core functions for digital I/O pin manipulation including set, clear, toggle, and read operations. The module manages GPIO pins for LEDs, relays, control signals, and input monitoring with parameter validation for safe operation.

int gpio_toggle_pin_ ( S_GPIO_Pin_TypeDef pin)

Toggle GPIO pin state.

Inverts the current state of the specified GPIO pin. If the pin is HIGH, it becomes LOW; if LOW, it becomes HIGH. Validates port and pin parameters before performing the operation.

Parameters
[in]pinS_GPIO_Pin_TypeDef structure containing port and pin information (e.g., GPIO_LED1, GPIO_MCU_LLC_RELAY).
Return values
SUCCESSPin toggled successfully.
ERRORInvalid parameter (NULL port or zero pin value).
Note
This function checks for valid port pointer and non-zero pin value. Commonly used for blinking LEDs or generating pulse signals.

Inverts the current state of the specified GPIO pin using HAL_GPIO_TogglePin function. If the pin is currently HIGH, it becomes LOW; if LOW, it becomes HIGH. Performs validation of port pointer and pin value before executing the operation.

Parameters
[in]pinS_GPIO_Pin_TypeDef structure containing GPIO port pointer and pin mask (e.g., GPIO_LED1, GPIO_MCU_LLC_RELAY). Structure is passed by value.
Return values
SUCCESSPin successfully toggled.
ERRORInvalid parameter detected - either port pointer is NULL or pin value is zero (invalid pin mask).
Note
This function validates both port and pin parameters. Port must be a valid GPIO peripheral pointer and pin must be a non-zero value. Commonly used for blinking LEDs or generating pulse signals. The toggle operation is atomic at hardware level.
