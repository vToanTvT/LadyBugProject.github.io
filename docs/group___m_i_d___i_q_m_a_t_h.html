Detailed Description

Fixed-point arithmetic operations and type definitions.

This module provides fixed-point (Q-format) math operations including Q15 and Q31 arithmetic. Defines types, conversion macros, and basic operations (add, subtract, multiply) optimized for embedded systems without hardware floating-point units. Supports signed/unsigned fixed-point representations for efficient control algorithms.

Fixed-point arithmetic operations in Q15 format.

Convert floating-point value to Q15 fixed-point format.

Q15 format: 1 sign bit + 15 fractional bits, range [-1, 1). Formula: Q15 = (float_value * 32768) - 0.5

Parameters
[in]aFloating-point value to convert (range: -1.0 to 0.99997).
Returns
Q31 representation of the value (stored as q31_t).
Note
Result is q31_t type but represents Q15 value (upper 16 bits unused).
